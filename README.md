# Buyocracy 🛒
### The Rule of Buyers, Not the Rule of Shops.

Buyocracy is a platform where buyers unite in groups to negotiate the best prices with sellers. By pooling our buying power, we flip the script on traditional retail.

## 🚀 Concept
- **United Demand:** Post what you want to buy.
- **Group Strength:** Others join your group for the same item.
- **Better Deals:** We negotiate bulk discounts directly with sellers.

## ✍️ Our Story
Founded by **Naitik**, a Class 8 student from DPS, Hyderabad. Buyocracy was built on the belief that community-driven commerce is the future.
